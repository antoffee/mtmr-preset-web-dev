# mtmr-preset-web-dev
Preset for web developers

MTMR (https://github.com/Toxblh/MTMR) preset containing
- volume controls
- ❤️ localhost select (3000, 8000, 6006)
- 💙 open VSCode button
- 💜 reload page button (cmd + shift + r) for Chrome
- 🤍 devtools button (cmd + shift + i)
- battery charge
- clock
- delete btn
